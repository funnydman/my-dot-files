# FlexiSpot E7 Plus — Memory Presets

## Control Panel Buttons

- **Up arrow (▲)** — raise the desk
- **Down arrow (▼)** — lower the desk
- **M** — memory/save button
- **1** — memory slot 1
- **2** — memory slot 2
- **Sitting person icon** — preset for seated height
- **Standing person icon** — preset for standing height

4 programmable memory slots total.

## How to Save a Height

1. Use **▲ / ▼** to move the desk to your desired height
2. Press the **M** button — the display will show **"S"**
3. While "S" is showing, press the button you want to save to (**1**, **2**, **sitting icon**, or **standing icon**)
4. The height is now stored

### Suggested setup

| Button              | Use                                                  |
|---------------------|------------------------------------------------------|
| Sitting person icon | Your seated height                                   |
| Standing person icon| Your standing height                                 |
| 1                   | Alternative seated height (e.g. different chair)     |
| 2                   | Alternative standing height or drafting-stool height  |

## How to Recall a Saved Height

Press the memory button (1, 2, sitting icon, or standing icon) and the desk will automatically move to the stored height. On some firmware versions you may need to double-tap or hold the button until the desk starts moving.

## Child Lock

- **Lock:** Hold **M** for ~5 seconds until the display shows **"LOC"**
- **Unlock:** Hold **M** again for ~5 seconds until the display returns to the numeric height

## Reset

If the desk behaves erratically:

1. Press **▼** until the display shows **"RST"**
2. Continue holding **▼** until the desk lowers slightly and rises back up
3. The desk is now recalibrated
